1. Snort
Algorithme :
But : Les joueurs placent leurs pions sur un plateau n*n de façon à bloquer l'adversaire.
Mouvement : À chaque tour, un joueur place un pion sur une case libre.
Fin de partie : La partie se termine quand plus aucun pion ne peut être placé. Le dernier joueur à avoir joué gagne.

2. Dodgem
Algorithme :
But : Chaque joueur doit déplacer ses pions de l’autre côté du plateau tout en bloquant les mouvements de l'adversaire.
Mouvement : Les joueurs déplacent leurs pions dans une direction (nord, sud, est, ou ouest) si la case est libre.
Fin de partie : Le jeu se termine lorsque l'un des joueurs ne peut plus déplacer ses pions ou lorsque tous ses pions ont atteint l'autre bord.
